# A字母来源

- 长的像牛角，表示尖。ang
- 26个字母排第一个，表示一 a,an
- A有向上的意思，表示加 add
- A有反向的意思，表示否定，happy,unhappy
- A有箭头的意思，表示方向, asleep(去睡觉，睡着)
- able:能力

- able,abl,abil (能力)。disable:失去能力
- ac：尖、酸
- acre（英亩），agri（耕田，农业）
- acid：尖、酸
- ac,anx:尖、酸
- am:爱
- ag,ig,act,urg:行动
- amb、ab：周围
- ani,anim:生命

- an,ant,anc:前、古老
- star,aster,astro:星星✨
